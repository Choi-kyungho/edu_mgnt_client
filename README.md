### Npm 버전 체크 및 재설치

- 명령어를 통하여 버전을 체크 합니다.

```
$ npm -version
```

만약 버전이 v6.14.15 이 아니라면 npm i -g npm@6.14.15 명령어를 통하여 재설치 해주세요.

- Yarn 설치

```
$ npm install -g yarn
```
